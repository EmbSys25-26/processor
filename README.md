# VGA protocol implementation

- Implements full VGA protocol and checks functional and timing requirements and constraints. Support info in is report and in powerpoint.

- Test is performed with Zybo Z7-10 switches:
  
	(From left to rigth)

	- 1st is Reset
	- 2ns is Red
	- 3rd is Green
	- 4th is Blue
	
	(Note: From left to right the priority decreases)

- Timing report is attached.

- Respective waveforms for each testbench are attached.

- Deals with CDC.
